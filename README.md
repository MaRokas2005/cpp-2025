# Pasirinkau daryti klasę gyvūnas pirmai užduočiai

Ištestuota objekto Animal įrašymas į failą ir nuskaitymas iš failo dvejetainiu formatu.

## Paleidimas
```bash
make
main.exe
```
## Dokumentacijos generavimas
```bash
make docs
```
## Išvalymas
```bash
make clean
```
## Išvalymas ir paleidimas iš naujo
```bash
make rebuild
main.exe
```
